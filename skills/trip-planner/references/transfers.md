# Transfer Reference Times

Estimated road transfer durations between airports and common destinations. Read only the regional section you need — Claude doesn't need to load every country.

Times assume normal traffic. Add ~30 min for peak hours (Istanbul rush, summer Friday evenings on Mediterranean coast). Cost estimates are for private taxi or shared shuttle, on the upper end — booking via the hotel is usually cheaper.

## How to handle non-listed routes

If the trip is in a region not covered above:

1. Default to a rough estimate from public sources (Google Maps "driving" + traffic for the day-of-week).
2. **Tell the user** the estimate is rough and that they should verify locally (hotel concierge, Wolt/Yango/Careem driver).
3. Add a P3 task to expand this file with the new country, including a fresh datapoint from the trip.

Updating policy: re-verify times in this file every ~6 months, since road construction (especially in Turkey and Egypt) changes them.
